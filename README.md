# L'ura (Lura)

World of Warcraft retail addon: a small bar of raid-marker buttons that send `/say` with a short label plus a brace tag (e.g. `T {star}`, `Circle {circle}`).

## Install

Copy the `Lura` folder into `_retail_\Interface\AddOns\` and enable **L'ura** in the AddOns list. `/reload` if needed.

## Usage

- **Options:** Esc → Options → AddOns → **L'ura**
- **Slash:** `/lura` (see `/lura help`)
- **Keybinds:** Esc → Options → Key Bindings → AddOns → **L'ura** (unbound by default)
